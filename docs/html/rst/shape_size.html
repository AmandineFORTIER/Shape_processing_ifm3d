shape_size.shape_size(contour, dist)¶

Get the shape size.

Get the height, width and length of the shape.

Parameters

  • contour (numpy.ndarray) – The object’s contours.

  • dist (numpy.ndarray) – The distance image

Returns

  • height (float) – The object height in meters

  • length (float) – The object length in meters

  • width (float) – The object width in meters
