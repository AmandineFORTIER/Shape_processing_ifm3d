Shape module¶

class Shape.Shape¶

Bases: enum.Enum

Enum to define the different types of shapes.

These shapes can be detected by this algorithm. ALL : detects all types of shapes.

PARTIAL : detects the shapes that are on the edge of the image.

UNKNOW : if it’s another type of shape.

CIRCLE = 1¶
ELLIPSE = 2¶
HEPTAGON = 8¶
HEXAGON = 7¶
OCTAGON = 9¶
PARTIAL = 10¶
PENTAGON = 6¶
RECTANGLE = 5¶
SQUARE = 4¶
TRIANGLE = 3¶
UNKNOW = 11¶
